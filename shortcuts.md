# Custom shortcuts

## System Preferences

### App Shortcuts
- `Alt-Cmd <-` Show Previous Tab
- `Alt-Cmd ->` Show Next Tab
- `Alt Cmd ,` System Preferences...

### Screenshots
- Fjerne `Ctrl` fra screenshot med kopiering

### Keyboard
- Move focus to next window `Cmd <`

## IntelliJ
- Kommentar: `Shift Cmd ,`
